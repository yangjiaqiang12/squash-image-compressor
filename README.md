## Why Squash?

Most online image compressors upload your files to their servers. Squash is different — **everything happens right in your browser**. Your images never leave your device. No privacy concerns, no waiting for uploads, no file size limits.

## Features

- 🔒 **100% Private** — All processing is client-side. Zero server uploads.
- ⚡ **Instant** — No network round-trips. Compression starts immediately.
- 🎨 **Multi-format** — JPEG, PNG, WebP, AVIF support with auto-detection.
- 📦 **Batch Processing** — Compress multiple images at once.
- 🎚️ **Quality Control** — Adjust compression from 1-100% with live preview.
- 📐 **Resize** — Set maximum dimensions while compressing.
- 🌓 **Dark Mode** — Automatic light/dark theme support.
- 📱 **PWA Ready** — Works on desktop, tablet, and mobile.
- 💰 **Free Forever** — No usage limits, no watermarks.

## Tech Stack

Vanilla HTML/CSS/JavaScript — no frameworks, no dependencies, no build step. Uses the Canvas API for image processing. Deploys instantly to any static host.

## Self-Host

```bash
git clone https://github.com/yangjiaqiang12/squash-image-compressor.git
cd squash-image-compressor
npx serve .
```

Or just open `index.html` in your browser. No server required.
